## 本地开发

```sh-session
git clone https://github.com/yesmore/blogv2.git
cd blogv2
```

复制文件 [`.env.example`](https://github.com/yesmore/blogv2/blob/main/.env.example) 并重命名为 `.env` .

然后安装依赖项并运行：
```sh-session
pnpm install
pnpm dev
```
